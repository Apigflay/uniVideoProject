用户行为规则之禁止虚拟币
和虚拟物品违规交易

  • 1、本用户行为规则为用户与本产品运营方签署的用户协议/服务协议或其他类似协议（统称为“服务协议”）的有效组成部分。
  • 2、服务协议中约定的用户应当遵守的行为规范仍然有效，包括但不限于：（1）禁止色情淫秽内容的传播；（2）用户不得参与赌博活动；（3）用户不得利用互联网从事危害国家安全、走私、毒品等违法犯罪活动；（4）禁止从事低俗或有违社会道德的网络表演。上述属于严重违反服务协议和违反法律的行为将受到严厉打击。
  • 3、本用户行为规则主要内容为（1）禁止违规兑换虚拟货币和礼物；（2）禁止用户通过非正规途径购买虚拟币及介入非法买卖交易。具体请阅读以下规则全文内容。
  • 4、本用户行为规则在公布之日起生效，并将由运营方适时调整。部分重要的行为规则，将由用户在线签署或线下签署，以便用户对行为规则引起重视。
  • 各方经协商一致，签订如下协议，共同遵照执行：

    第一条 禁止违规兑换虚拟货币和礼物

  1. 1.本产品包含的互动社交、娱乐服务，该等服务可能需要由用户以货币兑换产品虚拟币并使用虚拟币获得服务功能。用户兑换的虚拟币、礼物仅能用于正常服务功能，不得用于兑换现金、支持其他交易、支持违法行为等用途。
  2. 2.本产品如包含主播功能，则主播可依照产品规范获得主播报酬，主播报酬按照主播工作所获得的虚拟币、礼物的数量等标准进行衡量，由运营方或主播所属的工作室（或家族，以下含义同）支付主播报酬。主播、工作室不得为虚增业绩或其他目的，向任何用户提供返现或其他利益，不得为其他用户的虚拟币、礼物提供兑换现金的服务或成为其兑换通道。包括但不限于以下行为，均是禁止的：
    （1）与用户出现任何现金交易；
    （2）收到用户奖励礼物、宝宝等,私下返现金给用户；
    （3）与用户私下合作，以主播报酬名义申请报酬，获得报酬，但实质为将虚拟币、礼物兑换为现金的其他行为。
    （4）涉嫌网络赌博或其他违法活动的行为。
  3. 3.一经发现违规行为，运营方有权没收所有违规所得,封禁账号等处罚。
  4. 4.第一次违规的，扣罚一周所有家族收益。第二次违规，运营方有权直接关闭家族权限。
  5. 5.家族的主播参与违规行为的，家族室主承担连带责任，一并处罚。

    第二条 禁止用户通过非正规途径购买虚拟币及介入非法买卖交易

  1. 1.用户如需使用产品功能而需要充值虚拟币的，应当通过产品认可的经销商或自充途径获得, 不得与其他用户之间通过私下交易获得。主播、家族室主等全部用户均应当遵守。
  2. 2.主播、家族室主如收到用户虚拟币、礼物，私下返现金或其他利益给用户或者刷虚拟币给用户进行套现，均属于违规行为。
  3. 3.用户通过非正规途径获得的虚拟币或礼物，一经发现，运营方有权没收并对用户账户进行封禁或限制权限处理。
  4. 4.第一次违规的，扣罚一周所有家族收益。第二次违规，运营方有权直接关闭家族权限。
  5. 5.家族主播参与违规行为的，家族室主承担连带责任，一并处罚。
  6. 6.运营方对非正规的虚拟币交易渠道将进行严厉打击，如主播或家族室主参与其中的，一经发现没收违规所得,并将处以严厉罚款。
